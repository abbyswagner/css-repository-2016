How To Use Javascript

What Is Javascript and What Can You Create With It?

Javascript, not to be confused with Java, is a computer programming language. Unlike html, with Javascript you can make the website interactive. Javascript can read user inputed data and make calculations and/or actions. With this language, I made a calculator and a todo list. Read on to see how I used different Javascript elements.

Defining Variables

In Javascript, it becomes necessary to define variables in order to use them in the code. Additionally, you must connect the javascript variables to the html variables. While naming variables, you can do such by using the DOM element, or document.getElementById. In the provided code to the right, I create four different variables. Think of it like an equation. First, you list the new variable with the var element (for example var "firstNumber"). Next, equals the document.getElementById element. Lastly, put the id of what you want to link to the javascript page in quotations and parentheses. Also, do not forget the semicolons because this is how you list different elements in javascript. Also note the variables are any values you need for your function to work. In the example to the right, I define the variables I'll need for my calculator to work. I named the div carrying the user-inputed first number, second number, and operation as you need these two things to figure out the what the user wants the calculator to do in the function. I also defined the div carrying the answer because once the calculator finds the answer, it will have to display it in said div.

My Javascript Page Code: 

        
var firstNumber = document.getElementById("input1");
var operation = document.getElementById("operation");
var secondNumber = document.getElementById("input2");
var calculatorbutton = document.getElementById("calculatorbutton");

If Statements

One of the most useful things you can add to your function is an if statement. An if statement basically a way to make the computer do something given the right conditions. For example, if you wanted the background of your website to turn pink when the user entered the word "pink", you could use an if statement, the condition being the inputed string (set of characters) "pink" and the event would be the colour change. If statements start with an "if", the condition you want your operation to happen in in parentheses, and a set of curly brackets. Within those curly brackets is what you want your computer to do. This being said, you can also place an if statement within an if statement when you want an event to occur given two conditions, like the example to the right. In this example I say if the user inputed a plus sign in one of the drop downs in my calculator, the following if statements will come into play. The first of these if statements is if the user inputed numbers are "2" and "2", the answer (note it is "innerHTML" so that the computer displays the actual value) will be the string "fish". One small thing to note is some characters have special meanings ("&&" is "and" and "===" is "equals") and you put quotations around strings (which as mentioned before are lines of characters that are not numbers to be calculated).

Another thing about if statements is the first statement begins with "if" and all following statements start with "else if" with the exception being "else" statements who are without a specific characteristic, whose condition is just that it does not fall under any other categories. Else if statements are done the same way with "if" being replaced by "else if". Else statements, however, just have "else" and the proceeding curly brackets which just have what you want the computer to do. In my example, I use an else if statement to say when the user inputs "1234567890" for both numbers while using addition, the computer will display "I'm too lazy". Getting Your Operation To Work

You're almost ready to have a working function; this is the simplest part! To make your function work, just have the id of the button with ".addEventListener" and, within parentheses, "click" and the name of your function. You basically tell the computer when the button is clicked, to run the function.
